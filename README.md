# ML_June_26

House Price Prediction Using Linear Regression

--Week 1--

Project Description

This project is a simple Machine Learning application that predicts house prices using the Linear Regression algorithm. The project uses the California Housing Dataset available in Scikit-learn. Before building the model, Exploratory Data Analysis (EDA) is performed to understand the dataset and identify important factors that affect house prices.

Objective

The main objective of this project is to analyze housing data and build a Linear Regression model that can learn the relationship between different housing features and house prices. This project helps in understanding the basic workflow of a Machine Learning project, including data loading, data analysis, data preparation, and model training.

Tools and Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

Dataset Information

The California Housing Dataset contains information related to houses in California. The dataset includes various features such as median income, house age, average number of rooms, average number of bedrooms, population, occupancy, latitude, and longitude. These features are used to predict the house price.

Methodology

1. Import the required Python libraries.
2. Load the California Housing Dataset.
3. Perform Exploratory Data Analysis (EDA) by examining the dataset structure, summary statistics, and missing values.
4. Visualize the data using histograms and scatter plots.
5. Separate the input features and target variable.
6. Split the dataset into training and testing datasets.
7. Create and train a Linear Regression model.
8. Display the model coefficients to understand the impact of each feature on house prices.

Results

The Linear Regression model is successfully trained using the housing dataset. The model learns the relationship between housing features and house prices. The feature coefficients provide insights into how different factors influence house prices.
